# Presentation Runbook

## Preparacion

1. Ejecutar `npm run dev -- --host 127.0.0.1`.
2. Abrir `http://127.0.0.1:5173/` en Chrome o Edge actualizado.
3. Usar resolucion `1920x1080` o `1280x720`.
4. Mantener calidad `Alta` en el computador de presentacion; cambiar a `Equilibrada` si se perciben saltos.

## Recorrido Recomendado

1. Presentar el viaje luminoso y la propuesta de transporte especial.
2. Avanzar por confianza, flota y revelacion.
3. Mostrar el showroom y abrir dos hotspots.
4. Activar la vista interior conceptual y variar la luz.
5. Pulsar `Cotizar este bus`.
6. Completar el formulario y mostrar la recomendacion sin precio ficticio.
7. Cerrar con el convoy y el mensaje final.

## Contingencia

- Ante rendimiento insuficiente, seleccionar calidad `Ligera`.
- Ante fallo de WebGL, la demo activa automáticamente la vista 360.
- Ante un problema general, reproducir:
  `public/demo-assets/backup/cootrasec-demo-backup.mp4`.

## Mensajes Clave

- La demo comunica una propuesta digital, no disponibilidad ni precios reales.
- Los vehículos son conceptos genéricos sin marca.
- El sitio definitivo puede reemplazar los recursos demo sin cambiar la arquitectura.
